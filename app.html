Hit 100: A Coding Challenge

Always be at 100! Hit 100 is a Javascript coding challenge using loops. The objective is to write a program which displays the numbers 0 - 100, with the even mumbers in BOLD.

Enter a starting value and an ending value. Press the button to see the numbers appear on the page!
